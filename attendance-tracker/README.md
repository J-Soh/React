#### Attandance Tracker App

[![App Demo](https://img.youtube.com/vi/wVvXybZO05s/maxresdefault.jpg)](https://youtu.be/wVvXybZO05s)

- [Live build](https://nsf-attendance-tracker.netlify.app/)
- This project was developed using React (frontend) + Tailwind CSS, and Supabase (backend).
    - Tailwind CSS is used to create a responsive and structured user interface.

- This app allows users to record, update, and manage attendance records.
    - This allows the faster timely tracking of attandence in advance. 

- Added updated Features:
    - This app now allows you to extract `.txt` format information. 
    - Now adminstrator can add / delete / hold drag name card personals to change position via log in system. 
    - Improved Design for better visibility.

#### To run this project:

1. **Clone this repository** to your local machine:
   ```bash
   git clone https://github.com/J-Soh/React.git
   ```

2. **Install dependencies:**
- [nodejs](https://nodejs.org/en/download)
- react

3. **Navigate to root directionary:**

```
cd "attenance-tracker"
```

4. **Run this command with `Git-Bash`:**

```
npm install
npm start
```
